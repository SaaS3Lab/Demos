Game-AI Demo

    The Demo aims to train a Game Bot AI using Reinforcement Learning dAPI with the following steps:
    (1) You can create a dAPI by clicking the 'plus' icon and choosing a dAPI type. If you using a neural network, you should add at least one hidden layer with relu activation function.
    (2) Click the selection box to choose your just created dAPI.
    (3) Click 'play' button to train the Game Bot and the 'speed' button control the training speed.
    Your AI-bot will learn by itself and self-evolution.
